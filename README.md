# Дивошкола на GitHub
Сайт Дивошколи — це інформаційний продукт з відкритим вихідним кодом, опублікований на GitHub під ліцензією MIT. Ми хочемо, щоб знання були доступнішими, а процеси здобування освіти та оволодіння навичками — якіснішими. Ми відкриті до співпраці і будемо вдячні за будь-яку допомогу в внашій роботі.
